# 038 비로그인 체험 장부 리포트

## 목적

Supabase 테스트 계정이 아직 준비되지 않아도 사용자가 1억 챌린지의 핵심 재미를 더 깊게 체험할 수 있도록 했습니다.

기존에는 비로그인 상태에서 절약액 계산만 가능했습니다.
이제는 비로그인 상태에서도 기기 안에 임시 체험 기록을 저장할 수 있습니다.

## 추가한 기능

### 1. 방문 완료 체험 기록 저장

장소 상세에서 비로그인 상태로 `방문 완료`를 누르면 `방문 완료 체험` 시트가 열립니다.

이제 여기서 `체험 기록 저장` 버튼을 누르면 아래 정보가 로컬 기기에 저장됩니다.

- 장소명
- 메뉴명
- 예상 절약액
- 일반 기준가
- 실제 이용가
- 카테고리
- 저장 시각

### 2. 1억 챌린지 체험 화면에 누적 반영

MY 탭의 `1억 챌린지 체험 화면 보기`에서 로컬 체험 기록을 읽어 누적 절약액과 최근 기록을 보여줍니다.

### 3. 체험 기록 비우기

체험 기록이 있으면 `체험 기록 비우기` 버튼이 표시됩니다.

## 보안/운영 판단

이 기능은 Supabase DB에 저장하지 않습니다.
즉, 운영 데이터나 사용자 계정 데이터에 영향을 주지 않습니다.

로그인 전 전환율을 높이기 위한 로컬 체험 기능입니다.

## 남은 작업

향후 로그인 성공 후에는 아래 정책을 선택해야 합니다.

1. 로컬 체험 기록을 실제 계정으로 이전할지
2. 체험 기록은 안내용으로만 두고 실제 방문 기록은 새로 받게 할지

MVP에서는 2번이 더 안전합니다.
