# 041 로그인 진단 UX 개선 리포트

## 작업 목표

로그인 계정이 아직 정상화되지 않았을 때, 사용자가 앱 화면만 보고도 무엇이 문제인지 이해할 수 있게 만들었습니다.

## 현재 확인된 상태

외부 API 확인 결과, 테스트 계정 로그인은 아직 아래 오류를 반환합니다.

- 오류: `invalid_credentials`
- 의미: 이메일 유저가 Supabase Auth에 없거나, 비밀번호가 다르거나, 계정 Confirm 처리가 안 된 상태일 가능성이 높음

## 변경한 내용

### 1. 로그인 카드에 현재 진단 표시

MY 탭 로그인 카드 안에 다음 내용을 표시합니다.

- 현재 진단: 로그인 API가 `invalid_credentials` 반환 중
- 가장 흔한 원인: 유저 없음 또는 비밀번호 불일치
- 해결 기준: Supabase Users 목록에 유저가 있고 Confirmed 상태

### 2. 초보자용 다음 행동을 명확화

기존 복사 버튼과 체크리스트는 유지하면서, 왜 이 작업이 필요한지 더 쉽게 설명했습니다.

## 수정 파일
