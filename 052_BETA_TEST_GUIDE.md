## 1. 베타 테스터에게 안내할 내용

### 1.1 짠테크맵이란?

짠테크맵은 **저렴한 장소를 찾고, 가격을 제보하고, 절약액을 기록하는 앱**입니다.

- 🗺️ 내 주변의 저렴한 카페·식당·생활 서비스를 지도에서 찾을 수 있어요
- 💰 직접 방문한 곳의 가격을 제보하면 포인트를 받을 수 있어요
- 🏆 절약한 금액이 "1억 챌린지"에 누적되어 모두 함께 1억을 달성해요

### 1.2 베타 테스트 기간 및 일정

| 항목 | 내용 |
|------|------|
| 테스트 기간 | 2026년 5월 중 (약 2주간) |
| 참여 인원 | 10~20명 |
| 배포 방식 | TestFlight (iOS) |
| 소통 채널 | 카카오톡 오픈채팅방 (또는 디스코드) |

### 1.3 테스터 행동 수칙

1. ✅ **솔직하게** — 좋은 점과 불편한 점 모두 말씀해주세요
2. ✅ **자주 써주세요** — 매일 1회 이상 앱을 열어봐주세요
3. ✅ **버그는 즉시** — 이상한 동작을 발견하면 바로 제보해주세요
4. ✅ **실제처럼** — 실제로 방문하고 결제한 것을 제보해주세요
5. ❌ **허위 제보 금지** — 가짜 가격이나 방문 기록을 올리지 마세요
6. ❌ **외부 공유 금지** — 베타 앱 스크린샷을 SNS에 올리지 마세요

### 1.4 포인트 제도 안내

| 활동 | 포인트 |
|------|--------|
| 가격 제보 승인 | +10P |
| 가격 변동 제보 승인 | +15P |
| 방문 완료 | +5P |
| 7일 연속 방문 | +20P (보너스) |
| 버그 제보 | +20P |
| 피드백 설문 작성 | +30P |

## 2. 테스트해야 할 기능

### 2.1 핵심 기능 테스트 (필수)

| # | 기능 | 테스트 항목 | 우선순위 |
|---|------|-------------|----------|
| 1 | **회원가입/로그인** | 이메일 회원가입, 로그인, 로그아웃, 자동 로그인 | 🔴 최우선 |
| 2 | **지도 탐색** | 지도 로딩, 마커 표시, 장소 카드 클릭, 카테고리 필터 | 🔴 최우선 |
| 3 | **가격 제보** | 장소 선택 → 메뉴/가격 입력 → 사진 첨부 → 제출 | 🔴 최우선 |
| 4 | **방문 완료** | 장소 방문 → GPS 확인 → 방문 완료 버튼 → 절약액 표시 | 🔴 최우선 |
| 5 | **1억 챌린지** | 개인 절약액 확인, 전체 진행률 확인, 기록 목록 | 🟡 중요 |
| 6 | **마이페이지** | 내 제보 목록, 내 방문 기록, 포인트 현황 | 🟡 중요 |

### 2.2 상세 시나리오

**시나리오 A: 첫 사용자 플로우**
```
1. 앱 설치 → 2. 회원가입 → 3. 지도에서 주변 탐색 → 4. 장소 카드 클릭
→ 5. 상세 정보 확인 → 6. 앱 종료 후 재실행 → 7. 자동 로그인 확인
```

**시나리오 B: 가격 제보 플로우**
```
1. 로그인 → 2. 장소 선택 → 3. "가격 제보" 버튼 → 4. 메뉴명 입력
→ 5. 가격 입력 → 6. 사진 촬영/첨부 → 7. 제출 → 8. "검수 중" 상태 확인
```

**시나리오 C: 방문 완료 + 절약액 플로우**
```
1. 실제 장소 방문 → 2. 앱 열기 → 3. 해당 장소 선택
→ 4. "방문 완료" 버튼 → 5. GPS 확인 → 6. 절약액 표시 확인
→ 7. 1억 챌린지 화면에서 반영 확인
```

**시나리오 D: 오류/엣지 케이스**
```
- 인터넷 끊긴 상태에서 앱 사용
- GPS 꺼진 상태에서 방문 완료 시도
- 사진 없이 가격 제보 시도
- 동일 장소 연속 방문 완료 시도
- 극단적 가격 (0원, 999,999원) 입력
```

## 3. 피드백 질문

### 3.1 일일 피드백 (매일 저녁 발송)

> 아래 질문에 간단히 답변해주세요 (1~2줄이면 충분!)

1. 오늘 앱을 몇 번 열어봤나요?
2. 가장 좋았던 기능은?
3. 가장 불편했던 점은?
4. 이해가 안 되는 화면이나 버튼이 있었나요?

### 3.2 주간 피드백 설문 (주 1회)

| # | 질문 | 응답 형태 |
|---|------|-----------|
| 1 | 이 앱을 친구에게 추천하시겠습니까? (0~10점) | NPS 점수 |
| 2 | 가격 제보 과정이 쉬웠나요? (1~5점) | 리커트 척도 |
| 3 | 절약액이 정확하다고 느꼈나요? (1~5점) | 리커트 척도 |
| 4 | 지도에서 원하는 장소를 쉽게 찾았나요? (1~5점) | 리커트 척도 |
| 5 | 앱 속도는 만족스러웠나요? (1~5점) | 리커트 척도 |
| 6 | 가장 개선이 필요한 기능 1가지는? | 자유 서술 |
| 7 | 추가됐으면 하는 기능이 있나요? | 자유 서술 |
| 8 | 기타 하고 싶은 말 | 자유 서술 |

### 3.3 베타 종료 시 최종 설문

1. 전반적으로 만족하셨나요? (1~10점)
2. 정식 출시되면 계속 사용하실 의향이 있나요?
3. 이 앱의 가장 큰 가치는 무엇이라고 생각하시나요?
4. 가장 크게 개선해야 할 점은?
5. 베타 테스트 경험은 어떠셨나요?

## 4. 버그 제보 양식

### 제보 채널
- **1순위:** 카카오톡 오픈채팅방에 아래 양식으로 제보
- **2순위:** 구글 폼 링크 (추후 공유)

### 버그 제보 양식 (복사해서 사용)

```
🐛 [버그 제보]

■ 어디서: (어느 화면에서 발생했나요?)
예) 가격 제보 화면, 지도 화면, 마이페이지

■ 무엇을: (무슨 동작을 했나요?)
예) "제출" 버튼을 눌렀어요

■ 어떻게: (무슨 일이 일어났나요?)
예) 앱이 멈추고 아무 반응이 없어요

■ 원래는: (원래 어떻게 되어야 하나요?)
예) "제보 완료" 알림이 떠야 해요

■ 재현: (다시 해봐도 같은 문제가 생기나요?)
예) 네, 3번 다 같은 문제

■ 스크린샷/화면 녹화: (있으면 첨부)

■ 기기/OS: (예: iPhone 15, iOS 18.4)

■ 심각도:
  🔴 앱 사용 불가 (크래시, 무한 로딩)
  🟡 기능 오작동 (잘못된 결과, 오류 메시지)
  🟢 사소한 문제 (오타, 디자인 깨짐)
```

### 좋은 버그 제보 예시

```
🐛 [버그 제보]

■ 어디서: 가격 제보 화면
■ 무엇을: 사진을 3장 첨부하고 "제출" 버튼을 누름
■ 어떻게: 로딩이 계속 돌다가 "오류가 발생했습니다" 메시지 표시
■ 원래는: 제보가 완료되어 "검수 중" 상태로 바뀌어야 함
■ 재현: 사진 3장일 때만 발생, 1~2장은 정상
■ 기기/OS: iPhone 14 Pro, iOS 18.3
■ 심각도: 🟡
```

## 5. 주의사항

### 5.1 베타 앱 관련

| 주의사항 | 설명 |
|----------|------|
| ⚠️ **데이터 초기화 가능** | 베타 기간 중 데이터가 초기화될 수 있습니다. 이 점 미리 양해 부탁드립니다. |
| ⚠️ **버전 업데이트** | TestFlight로 새 버전이 올라오면 반드시 업데이트해주세요. |
| ⚠️ **동시 접속** | 베타 서버는 소규모이므로 간헐적 속도 저하가 있을 수 있습니다. |
| ⚠️ **개인정보** | 베타 기간 중 수집된 데이터는 테스트 목적으로만 사용되며, 정식 출시 전 삭제될 수 있습니다. |

### 5.2 가격 제보 관련

- 🚫 **허위 가격 제보 금지:** 실제 확인한 가격만 제보해주세요
- 🚫 **타인 사진 도용 금지:** 직접 촬영한 사진만 사용해주세요
- 🚫 **과장된 절약액 금지:** 실제보다 부풀린 절약액은 삭제됩니다

### 5.3 소통 관련

- 💬 질문·건의는 오픈채팅방에서 자유롭게!
- 💬 다른 테스터의 피드백에 "+1"로 공감 표시 환영
- 💬 개인적인 이야기보다는 **앱 관련 대화** 위주로 부탁드립니다
- 💬 욕설·비하 발언은 퇴장 사유입니다
