# 036 QA 계정 복사 도우미 리포트

## 목적

Supabase 대시보드에서 테스트 유저를 만들 때 이메일과 비밀번호를 직접 입력하다가 오타가 나면 로그인이 계속 실패합니다.

이번 단계에서는 MY 탭 로그인 카드 안에 QA 계정 정보를 바로 복사할 수 있는 버튼을 추가했습니다.

## 추가한 기능

MY 탭 로그인 카드의 QA 안내 박스에 아래 버튼이 추가됩니다.

- 이메일 복사
- 비밀번호 복사

버튼을 누르면 iOS 클립보드에 값이 복사되고, 짧은 성공 피드백이 표시됩니다.

## QA 계정 정보

- Email: `jjantechmap.qa@gmail.com`
- Password: 별도 보관된 QA 비밀번호 사용

## 기대 효과

1. Supabase Users에서 테스트 유저를 만들 때 오타를 줄입니다.
2. 로그인 실패 원인이 코드인지 계정 설정인지 더 빨리 구분할 수 있습니다.
3. 왕초보 사용자가 대시보드 작업을 따라 하기 쉬워집니다.
