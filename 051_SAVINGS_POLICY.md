# SAVINGS_POLICY — 절약액 산정 운영 정책

> **문서 버전:** v1.0-beta | **작성일:** 2026-05-09 | **담당:** Ops Agent / 관리자

---

## 1. 절약액 산정 원칙

### 기본 공식

```
절약액 = 일반가(기준가) − 실제가(제보가)
```

### 핵심 원칙

1. **보수적 산정:** 과장되지 않도록 항상 보수적으로 계산
2. **실증 기반:** 일반가·실제가 모두 객관적 근거 필요
3. **1건 1산정:** 한 번 방문 시 대표 메뉴 **1건**만 산정
4. **상한선 적용:** 1건당 절약액 상한 **50,000원**
5. **투명성:** 산정 근거(일반가 출처, 실제가 증빙)를 사용자에게 공개

## 2. 일반가 기준

### 산정 우선순위

| 우선순위 | 방법 | 설명 |
|----------|------|------|
| 1순위 | **앱 내 카테고리 평균가** | 동일 카테고리 승인 가격의 중앙값 (30건 이상) |
| 2순위 | **공공 데이터** | 한국소비자원·통계청 소비자 물가 데이터 |
| 3순위 | **관리자 수동 설정** | 위 데이터 없을 시 시장 조사 후 직접 설정 |

### 카테고리별 베타 초기 일반가

| 카테고리 | 대표 메뉴 | 일반가 | 출처 |
|----------|-----------|--------|------|
| 커피/카페 | 아메리카노 (ICE) | 4,500원 | 소비자원 평균가 |
| 한식 | 백반/김치찌개 | 9,000원 | 외식 물가 통계 |
| 분식 | 김밥 1줄 | 4,000원 | 시장 조사 |
| 편의점 | 삼각김밥 | 1,500원 | 편의점 정가 |
| 미용 | 남성 커트 | 16,000원 | 서울 평균 |
| 세탁 | 와이셔츠 드라이 | 3,500원 | 업계 평균 |

### 갱신 주기
- **베타 기간:** 관리자 수동 월 1회 검토
- **정식 출시 후:** 앱 데이터 기반 자동 갱신 (분기 1회)

## 3. 실제가 기준

### 인정 vs 불인정

| 상황 | 인정 | 설명 |
|------|------|------|
| 메뉴판 정상 판매가 | ✅ | 가장 기본적인 실제가 |
| 멤버십/상시 할인가 | ✅ | 누구나 가입 가능한 할인 |
| 런치 특선 등 시간대 할인 | ✅ | "런치 특선" 표기 필수 |
| 오픈 이벤트/기간 한정 할인 | ⚠️ 조건부 | 이벤트 기간 명시 필수 |
| 지인 할인/VIP 특별가 | ❌ | 일반 이용 불가 |
| 1회성 쿠폰 적용가 | ❌ | 재현 불가 |
| 포인트/적립금 차감 후 가격 | ❌ | 개인 자산 사용 |

### 검증 방법
- 첨부 증빙(메뉴판/영수증)의 가격과 입력 가격 일치 여부 확인
- 불일치 시 사진 기준으로 관리자가 수정 후 승인

## 4. 절약액 과장 방지

### 자동 검증 규칙

| 규칙 | 처리 |
|------|------|
| 절약액이 일반가의 **80% 이상** | 🚨 자동 플래그 → 수동 검수 |
| 1건당 절약액 **50,000원 초과** | 🚫 50,000원으로 자동 캡 |
| 동일 사용자 1일 합계 **100,000원 초과** | 🚨 자동 플래그 → 관리자 확인 |
| 동일 장소 1일 **3회 이상** 방문 | 🚫 3회차부터 불인정 |
| 실제가 **0원** 제보 | 🚨 무료 이벤트 여부 확인 |

### 과장 적발 시 처분

| 위반 | 처분 |
|------|------|
| 실수 오입력 | 관리자 수정 후 재승인 |
| 의도적 과장 1회 | 해당 절약액 삭제 + 경고 |
| 의도적 과장 2회+ | 전건 재검수 + 7일 정지 |
| 조직적 부정행위 | 계정 정지 + 테스터 자격 박탈 |

## 5. 방문 완료 인정 기준

### 필수 조건

| 조건 | 검증 방법 |
|------|-----------|
| **실제 방문** | GPS 반경 **200m 이내** (대형 시설 500m) |
| **당일 등록** | 방문일 **23:59**까지만 가능 |
| **동일 장소 1일 1회** | 같은 장소 하루 2회 이상 불가 |
| **결제/이용 증빙** (절약액 산정 시) | 영수증 또는 이용 사진 첨부 |

### GPS 불가 시
- 사진 증빙(영수증 + 매장 외관 사진)으로 대체 인정

### 보상
- 방문 완료 1건당 **5P**
- 7일 연속 서로 다른 장소 방문 시 **추가 20P**

## 6. 1억 챌린지 반영 기준

### 반영 조건
```
✅ 가격 제보 "approved" 상태
✅ 방문 완료 인정 상태
✅ 절약액 > 0원
✅ 절약액 ≤ 50,000원/건
✅ 1일 동일 장소 1회 이내
```

### 반영 금액
```
챌린지 반영 금액 = MIN(절약액, 50,000원)
```

### 집계 방식

| 구분 | 설명 |
|------|------|
| 개인 절약액 | 사용자별 승인 절약액 누적 합계 |
| 전체 절약액 | 모든 사용자 승인 절약액 합계 (1억 목표) |
| 집계 주기 | 실시간 (승인 즉시 반영) |

### 절약액 취소/환수

| 상황 | 처리 |
|------|------|
| 가격 제보 사후 반려 | 해당 절약액 차감 |
| 허위 방문 적발 | 절약액 차감 + 경고 |
| 사용자 탈퇴 | 개인 삭제, 전체 합계 유지 |

### 마일스톤 이벤트

| 달성 금액 | 이벤트 |
|-----------|--------|
| 100만 원 | 🎉 축하 배너 + 전원 50P |
| 1,000만 원 | 🎉 축하 알림 + TOP 10 공개 |
| 5,000만 원 | 🎉 특별 이벤트 (추후 결정) |
| **1억 원** | 🏆 챌린지 달성! 기념 이벤트 + 전원 보상 |
