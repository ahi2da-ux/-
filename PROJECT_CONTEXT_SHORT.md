# PROJECT_CONTEXT_SHORT

## 한 줄 설명

짠테크맵은 사용자가 주변의 저렴한 장소를 찾고, 가격을 제보하고, 절약액을 1억 챌린지에 기록하는 iOS 위치 기반 절약 앱이다.

## 현재 MVP 완성도

- 로컬 코드/빌드/실행 기준: 약 93%
- Supabase `028`, `029`, `030`, `031`, `032` 적용 + QA 로그인 성공 시 예상: 약 93%
- 가격 제보 승인 E2E 통과 시 예상: 약 94%

## 구현 완료 기능

- SwiftUI 기반 iOS 앱
- 네이버 지도 SDK 기반 지도 화면
- 탐색 리스트, 지도 탐색, 장소 상세
- Supabase 장소/메뉴 조회
- Supabase Auth 로그인/회원가입/세션 저장
- 가격 제보 및 사진 첨부
- 관리자 검수 UI
- 포인트 장부
- 즐겨찾기
- 가격 알림 설정/이벤트 일부
- 1억 챌린지 요약, 절약 기록, 취소, 공유 카드 초안
- QA/운영자 내부 UI 숨김 플래그

## 현재 최대 병목

- QA 계정 로그인 `invalid_credentials`
- Supabase SQL `028`, `029`, `030`, `031`, `032` 실제 적용 여부 확인 필요
