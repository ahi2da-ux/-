# 030 MVP QA 체크리스트

짠테크맵 시뮬레이터 테스트를 위한 왕초보용 체크리스트입니다.

## 1. 테스트 계정 준비

1. Supabase 대시보드로 이동합니다.
2. Authentication > Users 메뉴를 엽니다.
3. Add user로 테스트 계정을 만듭니다.
4. 이메일 인증이 켜져 있다면 Confirm 처리를 합니다.
5. 앱 MY 탭에서 테스트 계정으로 로그인합니다.

## 2. 1억 챌린지 핵심 루프

1. 지도 탭 또는 탐색 탭에서 장소를 하나 선택합니다.
2. 장소 상세 화면 하단의 방문 완료 버튼을 누릅니다.
3. 메뉴를 선택하고 절약 기록을 저장합니다.
4. MY 탭의 1억 챌린지 카드에서 누적 절약액이 증가했는지 확인합니다.
5. 1억 챌린지 상세 화면에서 최근 절약 기록이 보이는지 확인합니다.

## 3. 중복 방지 QA

1. 같은 장소의 같은 메뉴를 다시 방문 완료 처리합니다.
2. 2시간 제한 메시지가 나오는지 확인합니다.
3. 앱이 멈추지 않고 안내 메시지만 표시되면 정상입니다.

## 4. 취소 QA

1. 1억 챌린지 상세 화면에서 최근 기록의 취소 버튼을 누릅니다.
2. 확인창에서 계속 보관을 누르면 아무 변화가 없어야 합니다.
3. 다시 취소 버튼을 누르고 취소하기를 선택합니다.
4. 누적 절약액이 줄고 최근 기록에서 사라지면 정상입니다.

## 5. 가격 제보 승인 QA

1. 로그인한 상태에서 장소 상세 > 가격 제보를 제출합니다.
2. 운영자 계정으로 MY 탭에 진입합니다.
3. 운영자 검수 카드에서 해당 제보를 승인합니다.
4. 포인트 장부, 가격표, 1억 챌린지, 가격 알림 연결 상태를 확인합니다.
5. 승인 파이프라인 QA가 정상으로 표시되면 통과입니다.

## 6. 운영자 복구 QA

1. 운영자 검수 카드의 승인 파이프라인 QA 섹션을 확인합니다.
2. 문제가 있는 항목이 있으면 이 항목 복구 버튼을 누릅니다.
3. 확인창에서 복구 실행을 선택합니다.
4. 다시 조회했을 때 정상 상태로 바뀌면 통과입니다.

## 현재 기준

- 1억 챌린지 핵심 루프: 약 91%
- 가격 제보/포인트/챌린지 연결: 약 92%
- 운영 QA/복구 도구: 약 88%
- 전체 MVP 완성률: 약 79%
