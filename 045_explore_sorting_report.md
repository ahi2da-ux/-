# 045 탐색 리스트 정렬 기능 추가 리포트

## 작업 목표

탐색 탭에서 사용자가 저렴한 장소를 더 빠르게 찾을 수 있도록 정렬 기능을 추가했습니다.

## 변경 전

탐색 탭에는 검색과 기본 필터가 있었지만, 사용자가 결과를 어떤 기준으로 정렬할지 명확히 선택하기 어려웠습니다.

## 변경 후

탐색 탭에 정렬 바를 추가했습니다.

정렬 옵션:

- 추천순
- 가격 낮은순
- 거리순
- 평점순

필터 옵션:

- 1만원 이하
- 5천원 이하
- 인증됨
- 영업중

## 정렬 기준

### 추천순

1. 추천 장소 우선
2. 신뢰도 높은 장소 우선
3. 가격 낮은 장소 우선

### 가격 낮은순

기본 가격이 낮은 장소부터 표시합니다.

### 거리순

실시간 위치가 있으면 현재 위치 기준으로 정렬합니다.
위치가 없으면 기존 거리 텍스트 기준으로 정렬합니다.

### 평점순

평점이 높은 장소부터 표시합니다.
평점이 같으면 리뷰 수가 많은 장소를 먼저 보여줍니다.

## 수정 파일

- 기존 검색/필터와 함께 동작
