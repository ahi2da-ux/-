## 실패 케이스

| 케이스 | 기대 결과 |
|---|---|
| 비로그인 상태에서 제보 | 로그인 유도 |
| 비로그인 상태에서 절약 기록 | 로그인 또는 체험 기록 안내 |
| 사진 업로드 실패 | 앱 크래시 없이 오류 안내 |
| Supabase 요청 실패 | 오류 메시지와 재시도 가능 |
| 위치 권한 거부 | 지도 기본 탐색 유지 |
| 같은 장소 연속 절약 기록 | 중복 제한 안내 |
| 일반 사용자가 관리자 RPC 호출 | 권한 오류 |

## 베타 통과 기준

- P0 시나리오 모두 통과
- 앱 크래시 없음
- QA 로그인 성공
- 가격 제보 승인 E2E 1회 이상 성공
- 실기기 사진/위치 권한 테스트 통과
