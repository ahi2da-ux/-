# 034 1억 챌린지 체험 모드 리포트

## 목적

Supabase 테스트 계정이 아직 로그인되지 않는 상태에서도 1억 챌린지 화면 구성을 시뮬레이터에서 확인할 수 있게 했습니다.

실제 저장/취소 QA는 로그인 후 진행해야 하지만, 화면 UX 검증은 더 이상 로그인에 막히지 않습니다.

## 추가한 기능

MY 탭의 1억 챌린지 카드에서 비로그인 상태일 때 아래 버튼이 보입니다.

`1억 챌린지 체험 화면 보기`

이 버튼을 누르면 샘플 데이터 기반의 체험 화면으로 이동합니다.

## 체험 화면에서 확인 가능한 것

1. 1억 챌린지 상단 헤더
2. 누적 절약액
3. 1억까지 남은 금액
4. 진행률 바
5. 현재 등급
6. 등급 단계 카드
7. 이번 달 절약액
8. 최근 절약 로그
9. 공유 카드 미리보기

## 체험 화면에서 하지 않는 것

- Supabase DB 저장
- 실제 로그인 세션 사용
- savings_logs insert
- profiles.current_savings 업데이트
- 절약 기록 취소 RPC 호출
